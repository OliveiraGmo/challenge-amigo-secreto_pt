Amigo Secreto
Este projeto é uma aplicação web simples para realizar sorteios de amigo secreto. Permite adicionar nomes de participantes e realizar o sorteio de forma aleatória, exibindo os resultados na tela.

Funcionalidades
Adicionar participantes: Insira os nomes dos participantes em um campo de texto e clique em "Adicionar" para incluí-los na lista.
Sorteio: Clique em "Sortear amigo" para realizar o sorteio aleatório e exibir os pares de amigo secreto na tela.
Validação: O sistema verifica se há pelo menos dois participantes antes de realizar o sorteio e se o campo de nome está preenchido.
Listagem de amigos: Mostra todos os amigos adicionados em uma lista.
Como usar
Clone ou baixe este repositório.
Abra o arquivo index.html em seu navegador.
Insira os nomes dos participantes no campo de texto e clique em "Adicionar".
Clique em "Sortear amigo" para ver os resultados do sorteio.
Estrutura do projeto
index.html: Estrutura HTML da página web.
app.js: Lógica JavaScript da aplicação.
style.css: Estilos CSS da página web.
assets/: Pasta com imagens utilizadas no projeto.
